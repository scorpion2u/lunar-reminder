# lunar-reminder
# 🌙 农历提醒 · 初一十五买水果

一个简单好用的农历提醒工具，专门为初一十五买水果、上香而设计。

## ✨ 功能特点

- ✅ **准确的农历日历** - 内置1900-2100年农历数据，无需网络
- ✅ **初一十五提醒** - 默认开启，提前3天连续提醒
  - 📅 后天初一 · 记得买水果
  - ⏰ 明天初一 · 记得买水果  
  - 🙏 今天初一 · 供水果上香
- ✅ **自定义提醒** - 支持添加其他农历/公历提醒
- ✅ **浏览器通知** - 每天自动弹出提醒
- ✅ **本地存储** - 所有设置保存在浏览器中
- ✅ **离线可用** - 不依赖任何网络API

## 🖥️ 使用方法

1. 打开网页即可使用
2. 点击开关可以开启/关闭初一十五提醒
3. 点击下方"添加自定义提醒"设置其他日期
4. 点击右上角菜单管理提醒和开启通知

## 📱 手机适配

完美支持手机浏览，界面简洁易用。
